Prosty plugin do blokowania użytkowników w WooCommerce na podstawie adresów e-mail, imion i nazwisk, numerów telefonów, domen oraz adresów IP.

== Description ==

WooCommerce Block Users pozwala na łatwe blokowanie użytkowników w WooCommerce, zapobiegając niechcianym zakupom lub zamówieniom od konkretnych klientów.

**Główne cechy:**
- Możliwość blokowania użytkowników na podstawie:
  - Adresu e-mail
  - Imienia i nazwiska
  - Numeru telefonu
  - Domeny e-mail
  - Adresu IP
- Łatwa integracja z ustawieniami WooCommerce.

== Installation ==

1. Prześlij folder `woocommerce-block-users` do katalogu `/wp-content/plugins/`.
2. Aktywuj plugin przez menu 'Wtyczki' w WordPress.
3. Przejdź do ustawień WooCommerce i skonfiguruj opcje blokowania według własnych potrzeb.

== Frequently Asked Questions ==

= Czy mogę zablokować użytkownika tylko na podstawie jego imienia? =

Tak, plugin umożliwia blokowanie użytkowników na podstawie różnych kryteriów, w tym imienia.

= Czy istnieje możliwość zablokowania wszystkich e-maili z konkretnej domeny? =

Tak, wystarczy dodać nazwę domeny w odpowiednim polu w ustawieniach pluginu.
